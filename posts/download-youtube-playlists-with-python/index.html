Download Youtube Playlists With Python

December 24, 2021 · Harry Zhou

Recently, I got a mp3 player from my dad so that I can listen to music in bed without using any devices that could potentially be distracting. However, most of the music on the mp3 player don’t particularly fit my taste, so I want to download my own Christmas playlist.

But my dad refused to tell me how he pirates his music. Now in the past, I’ve been using this command-line tool youtube-dl to download youtube videos and convert them to mp3. However, there are a few issues with the tool. First of all, it can be extremely slow sometimes. Secondly, I frequently received error messages like ERROR: unable to download video data: HTTP Error 403: Forbidden in the middle of downloading a playlist/album. Turns out that the way to fix this problem is to remove the cache using flags such as --rm-cache-dir. However, when I try to download an entire playlist, I don’t know how to add the flag.

Therefore, after a little research, I decided to go with a new method using the python library pytube. The following function would take the url of a video and convert it to mp3 and save it in a specified directory:

def download_single_video(url):
    yt = YouTube(url)

    # extract only audio
    video = yt.streams.filter(only_audio=True).first()

    # check for destination to save file
    destination = './Music/Relaxing2'

    # download the file
    out_file = video.download(output_path=destination)

    # save the file
    base, ext = os.path.splitext(out_file)
    new_file = base + '.mp3'
    os.rename(out_file, new_file)

To download an entire playlist, we just need the Playlist feature from the pytube library and iterate though each url in the playlist:

playlist = Playlist(
	str(input("Enter the URL of the video you want to download: \n>> ")))

print('Number of videos: %s' % len(playlist.video_urls))

for url in playlist.video_urls:
    download_single_video(url)

Using this method, I easily downloaded my amazing playlist. Not only did it work flawlessly, the new method was also way faster than using youtube-dl.
